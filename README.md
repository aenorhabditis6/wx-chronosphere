# wx-chronosphere

Abstract: A 3D interactive orrery and calendar system visualizing the "Integral Genesis" cosmology using Three.js and Google Gemini.

Features

3D Orrery: Fully rendered solar system with Kepler-inspired orbital mechanics and time-freezing interaction.

Higgs Field: Interactive background particle system simulating fluid physics and Standard Model lore.

AI Oracle: Integration with Gemini API for procedural event generation, natural language parsing, and real-time Earth data.

Persistence: Local storage support with JSON export functionality for permanent timeline management.
